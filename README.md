
ActLocalizer
===============
RainbowMage  
https://github.com/RainbowMage/


概要
---------------
ローカライズします。


ダウンロード
---------------
右側の「Download ZIP」を押してダウンロードします。  
gitクライアントを持ってる方は普通にクローンします。


ビルド方法
---------------
.NET Framework 4.5.1 がインストールされているコンピュータで make.bat を実行します。  
Client Profile 版にはビルドに必要なツールが含まれていないので注意。


使用方法
---------------
* ACTの作業フォルダに「locale」という名前のフォルダを作り、その中にTranslationsフォルダの中にある「ja-JP」フォルダをコピーします。  
* RainbowMage.ActPlugin.dll を ACTのプラグインとして読み込ませます。  
* プラグインを追加すると、Plugins タブの中に「Localizer」のタブページが出現します。
* Localeをja-JPに設定して再起動します。
  
UIのフォントは「メイリオ, 8」がオススメです。これ以上大きいとたぶん色々とはみ出します。
